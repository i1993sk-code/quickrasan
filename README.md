# QuickRasan 🛵

Blinkit-style grocery delivery for Ara, Bihar. 10-minute delivery of daily essentials.

## Tech Stack
- **Frontend:** React 19, Redux Toolkit, Tailwind CSS, Vite
- **Backend:** Node.js, Express, MongoDB (Mongoose)
- **Payments:** Stripe, Razorpay
- **Media:** Cloudinary

## Domain
- `quickrasan.com` and `quickrasan.in`

## Deploy
- Frontend: Vercel
- Backend: Render
